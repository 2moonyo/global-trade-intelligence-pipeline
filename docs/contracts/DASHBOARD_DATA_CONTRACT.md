# Dashboard Semantic Mart Data Contract

This contract describes the dashboard-facing dbt semantic marts currently defined under `models/marts/semantics`.

Active serving target:
- BigQuery marts schema generated by dbt from the `bigquery_dev` profile target.
- Looker Studio, or an equivalent BI layer, should connect to these semantic marts instead of raw silver tables.

The contract intentionally reflects the current dbt models and local schemas.

## Live Dashboard

Access the interactive Looker Studio dashboard here:

[Global Trade Intelligence Dashboard](https://datastudio.google.com/reporting/42e61127-a852-4663-b3e7-fb78c2858159)

## 1. `mart_dashboard_global_trade_overview`

### Purpose
Page 1 executive overview of reporter-level monthly trade activity and reporting completeness.

Answers:
- Who is trading the most?
- Is the latest reporting month complete?
- Which reporters are missing from the latest month?
- How is trade changing month to month?

`reporter_country_code + month_start_date`

- `latest_month_flag`
- `latest_complete_month_flag`
- `complete_month_flag`
